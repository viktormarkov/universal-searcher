### Launch
1) Build extension with ```yarn build```
2) Go to [chrome://extensions/](chrome://extensions/)
3) Enable "Developer mode"
4) Click on "Load unpacked"
5) Select ```dist``` directory

### P.S.
- ```yarn watch``` - watch and rebuild code on file changes
- ```yarn test``` - run tests
